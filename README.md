Favel's Bakery & Paws Café

This is the very first website I’ve ever built!
[Click here to see the live website!] (https://osher-amanda.github.io/My-Bakery-Project/)

It’s a landing page for Favel's Bakery & Paws Café, a cozy spot designed for both people and their pets. The idea was to create a digital space that feels as welcoming as the café itself, featuring artisan pastries for humans and a specialized "Pet Menu" for their four-legged friends.

The Concept

The site is a clean, one-page experience. I wanted to put a unique twist on the classic bakery project by making it a space for both people and pets. As a huge animal lover with four dogs of my own, creating a 'Paws Café' felt like the perfect way to bring my own personality into my first build.  I focused on a modern, "sweet" aesthetic using a soft pink color palette and high-quality photography. Since I’m just starting out, my main goal was to make sure the brand's "Paws Café" vibe came through clearly while keeping the technical side functional and organized.


Features

Dual Menu: I set up separate sections for human treats (pastries, coffee) and pet treats (pupcakes, dog biscuits) using interactive cards.

Responsive Grid: Using the Materialize grid system, the site automatically adjusts from a multi-column desktop view to a single-column mobile view.

Photo Gallery: I used a "material box" effect so users can click on the café photos to see them full-screen.

Contact System: There’s a functional form at the bottom connected to Formspree, so the café can actually receive inquiries from customers.

The Tech Side

I used HTML5, CSS, and JavaScript, specifically leveraging the Materialize framework to handle the heavy lifting for the navigation and responsiveness. One of the biggest challenges was getting the form to submit correctly (not a big challenge though and I managed to overcome it within minutes, Baruch Ha Shem), which helped me understand how front-end code interacts with back-end services.


Created as part of my first web development project at Developers Institute.
